Twid includes simple functions about Taiwan Identification Card system.
See also: http://en.wikipedia.org/wiki/National_Identification_Card_%28Republic_of_China%29

Quick Start
^^^^^^^^^^^
::

 >>> import twid
 >>> twid.female()
 'C290074048'
 >> twid.kaohsiung()
 'E278429854'
 >>> twid.kaohsiung(twid.male())
 'E104149003'
